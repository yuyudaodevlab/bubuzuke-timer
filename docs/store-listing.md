# Chrome ウェブストア掲載用テキスト

Chrome ウェブストアの「アイテムの詳細」入力欄にそのまま貼り付けられるよう、項目ごとに分けています。

> **注意（キーワードスパム対策）**
> ストアの説明文には、対応サービスの固有名（SNS名・ショッピングサイト名など）を列挙しないでください。
> 過去の審査で「過剰かつ/または無関係なキーワード（キーワードスパム）」として不承認になったため、
> 説明文では「SNS」「動画サイト」「ショッピングサイト」「通販サイト」といった一般的な表現のみを使用します。
> （対応サイトはユーザーがインストール時の権限一覧で確認できます）

## 権限の説明（審査用メモ）

ストア審査で権限の使用目的を聞かれた場合の説明文です。

```
- alarms: 定期的な巡回（使用時間の計測・催促タイミングの判定）のために使用します。
- notifications: 30分ごとのデスクトップ通知を表示するために使用します。
- storage: 設定・統計データをブラウザ内にのみ保存するために使用します（外部送信なし）。
- tabs: アクティブタブのURLを見て、監視対象サイト（SNS・ショッピングサイト）かどうかを
  判定するために使用します。閲覧内容の送信・記録は行いません。
- idle: ユーザーが離席中は使用時間としてカウントしないために使用します。
- ホスト権限（x.com, youtube.com など）: ページ内ポップアップ（催促画面）を表示するために
  必要です。ページの内容を読み取ったり外部送信したりすることはありません。
```
