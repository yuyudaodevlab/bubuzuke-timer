# SNSやりすぎ京都はんなり見張り番 ⛩️

SNS・ショッピングサイトの見すぎを、**京都弁の皮肉**でやんわり（？）注意してくれる Chrome 拡張機能どす。

> 「まあ、ようけタイムライン見はりますなあ。お仕事のほうはよろしいん？」

30分・60分と使い続けるほど、催促のセリフはだんだん棘を増していきます。
皮肉レベルは3段階 —— **はんなり** → **じわじわ** → **ぶぶ漬け**（意味、お分かりどすな？）
各サイトであった京都弁で催促するどす
※エセ京都弁なので正しい表現になっていない場合があります
パロディですのでその点ご了承ください！

## 📸 機能

### 監視対象サイト

| SNS | ショッピング |
|---|---|
| X (Twitter) | BOOTH |
| YouTube | Amazon (co.jp / com) |
| Instagram | 楽天市場 |
| TikTok | |

### 催促のしくみ

- ⏰ **30分ごと**: デスクトップ通知で京都弁の皮肉が届きます
- 🖼️ **60分ごと**: 見ているページの真ん中に京都モチーフのポップアップが出ます
  - 「おおきに、作業に戻ります」… 連続使用カウントをリセット
  - 「…あと5分だけ堪忍」… いったん閉じるだけ（また来ます）
  - 「今日はもう通知せんといて」… **今日一日通知しない**（日付が変わると自動解除）
- 連続使用時間は、対象サイトから **5分以上離れると自動リセット** されます
- PCの前にいない間（アイドル状態）はカウントされません

### セリフについて

- **各サイト専用のセリフ**を用意しています（Xにはエゴサの皮肉、Amazonには段ボールの城の皮肉…など）
- 催促された回数が増えるほど、皮肉レベルが自動で上がります
  - レベル1「はんなり」（1回目〜）
  - レベル2「じわじわ」（3回目〜）
  - レベル3「ぶぶ漬け」（6回目〜）

### 設定・確認できること（ツールバーのポップアップから）

- 🎴 **皮肉ガチャ** … 京都弁の皮肉をランダムに一言引けます
- 📊 **今日のお目付け帳** … 各サイトの使用時間と催促回数
- 📅 **七日間のふりかえり** … 直近1週間の使用時間グラフ
- 🔔 **通知設定** … 30分通知 / 60分ポップアップの ON・OFF
- 🔕 **今日一日通知しない** … ON で今日は静かに。OFF に戻せばいつでも解除
- 🛠️ **開発モード** … ON にすると下の「開発の間」が開きます

### 開発モード（開発の間）

設定の「開発モード」をONにすると使える、開発・動作確認向けの機能です。

- ⏱️ **通知の間隔** … 通知を出す間隔を 1〜480分 の範囲で変更できます（既定: 30分）
- ⏱️ **ポップアップの間隔** … ポップアップを出す間隔を 1〜480分 の範囲で変更できます（既定: 60分）
- 🐇 **テストモード** … ONにすると 30秒ごとに10分ぶん 時間が進みます（実時間の20倍速）。
  既定設定なら、実時間およそ1分半で30分通知、3分で60分ポップアップを確認できます。
  確認が終わったらOFFに戻すのをお忘れなく（使用時間の記録も20倍速で増えます）
- 🔔 **テスト通知** … 押した瞬間にランダムなセリフで通知を出します
- 🖼️ **テストポップアップ** … 押した瞬間にポップアップを出します
  （監視対象のサイトを開いたタブで試してください）

### おまけ機能

- ツールバーのアイコンに **今日の合計使用時間（分）** がバッジ表示されます
- 使用時間が少ない日は、京都弁で **褒めてくれます**（「今日はえらい真面目にしてはりますなあ」）
- ポップアップのフッターの一言も、使いすぎ具合で変わります

## 🔧 インストール方法
### Claudeウェブストアを使用する方法
https://chromewebstore.google.com/detail/jdplceibhnfpfoefeomoeppfhdmlmebc?utm_source=item-share-cb

### デベロッパーモードで読み込む方法。

1. このリポジトリをダウンロードします
   ```bash
   git clone https://github.com/yuyudaodevlab/SNS-yarisugi.git
   ```
   （または GitHub の「Code」→「Download ZIP」でダウンロードして解凍）

2. Chrome で `chrome://extensions/` を開きます

3. 右上の **「デベロッパーモード」** をONにします

4. **「パッケージ化されていない拡張機能を読み込む」** をクリックします

5. 手順1でダウンロードした `SNS-yarisugi` フォルダ（`manifest.json` があるフォルダ）を選択します

6. ツールバーに ⛩️ のアイコンが出たら完了どす。おきばりやす。

> **Note**: Microsoft Edge・Brave など Chromium 系ブラウザでも同じ手順で動きます。

### 通知が出ないときは

- OSの通知設定で Chrome の通知が許可されているか確認してください
  - **macOS**: システム設定 → 通知 → Google Chrome
  - **Windows**: 設定 → システム → 通知 → Google Chrome
- ポップアップの「30分ごとの通知」がONになっているか確認してください
- 「今日一日通知しない」がONのままになっていないか確認してください

## 📁 ファイル構成

```
SNS-yarisugi/
├── manifest.json          # 拡張機能の定義 (Manifest V3)
├── background.js          # 使用時間の計測・通知の判断（サービスワーカー）
├── content/
│   └── content.js         # ページ内ポップアップの表示
├── popup/
│   ├── popup.html         # ツールバーポップアップ（設定・統計）
│   ├── popup.css          # 京都モダンなスタイル
│   └── popup.js           # ポップアップのロジック
├── shared/
│   ├── sites.js           # 監視対象サイトの定義
│   └── phrases.js         # 京都弁セリフ集（サイト別 × 皮肉レベル3段階）
└── icons/                 # 鳥居と月のアイコン
```

## 🔒 プライバシー

- 使用時間などの記録はすべて **ブラウザ内（`chrome.storage.local`）にのみ保存** されます
- 外部サーバーへの通信は一切ありません
- 31日より古い記録は自動で削除されます
